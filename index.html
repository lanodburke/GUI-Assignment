What is Linux?

Linux is a Unix-like operating system that was designed to provide personal computer users a free or very low-cost operating system comparable to traditional and usually more expensive Unix systems. Linux has a reputation as a very efficient and fast-performing system. Linux's kernel (the central part of the operating system) was developed by Linus Torvalds at the University of Helsinki in Finland. To complete the operating system, Torvalds and other team members made use of system components developed by members of the Free Software Foundation for the GNU Project.

WHAT LINUX HAS TO OFFER


POWERFUL

Linux is an extremly powerful operating system that offers a wide variety of tools for users to develop and change the system to suit their own needs.

FAST

Along with being powerful, linux is an insanely fast operating system when using the linux terminal to carry out tasks.

LIGHT WEIGHT

Not like traditional operating systems linux is a very light weight operating system, depending on the linux distribution on average linux operating systems only take upto 3-5gb of space.



GREAT FOR DEVELOPERS

As linux is an open source OS it is great for developers, it lets developers design and customize the system to their needs.

USER FRIENDLY

Nowadays most linux distributions come with built in GUI's(Graphical User Interface) for new users to be able to use linux like any other Operating System.

EASY TO INSTALL

Very easy to install, you just find the distribution you want download it, copy it onto a flash drive and plug it into the device you want to boot the OS on.
